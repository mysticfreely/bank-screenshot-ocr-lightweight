百度OCR API（推荐）

  • 访问 百度智能云OCR 获取API密钥
  • 成本低，中文识别效果好

Azure OCR API

  • 访问 Azure认知服务 获取密钥
  • 企业级稳定性
  • 支持多种语言

Google Vision API

  • 访问 Google Cloud Vision 获取API密钥
  • 识别精度最高
  • 支持多种图像格式
